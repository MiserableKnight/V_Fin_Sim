# ✈️ 越捷航空油价冲击财务决策模型

VietJet Air Oil Shock Financial Decision Model

基于2025经审计财报附注28/36/31校准的短期油价冲击下的航班优化决策模型

---

## 🧮 数学模型

### 问题定义

面对油价外生冲击，航空公司需要回答三个核心问题：

1. **盈利性判断**：当前运营条件下能否实现盈利？
2. **边际决策**：是否应该削减航班？削减多少？
3. **策略评估**：票价调整和套期保值如何影响决策？

---

### 符号定义

#### 基础财务参数（财报校准值）

| 符号 | 含义 | 数值 | 数据来源 |
|------|------|------|----------|
| $R_{base}$ | 基础运营收入（剔除非航线主业收入） | 64.94 万亿₫ | 2025财报附注28 |
| $V_{fuel}$ | 基础燃油成本 | 24.70 万亿₫ | 2025财报附注36 |
| $V_{other}$ | 其他变动成本（外部服务费、变动人工、航路起降费等） | 22.70 万亿₫ | 2025财报附注36 |
| $FC$ | 固定成本（租赁费、折旧、固定人工、坏账等） | 19.06 万亿₫ | 2025财报附注36/31 |

#### 外生冲击变量（决策输入）

| 符号 | 含义 | 取值范围 |
|------|------|----------|
| $\Delta P_{oil}$ | 油价涨幅 | $[-1, +\infty)$ |
| $\Delta P_{fare}$ | 票价调整幅度 | $[-1, +\infty)$ |
| $h$ | 燃油套期保值覆盖率 | $[0, 1]$ |

#### 模型参数

| 符号 | 含义 | 取值 | 依据 |
|------|------|------|------|
| $\varepsilon$ | 需求价格弹性 | -0.8 | 行业研究文献 |
| $\alpha$ | 网络骨架保留权重 | 0.25 | 航空运营实践 |

---

### 核心方程

#### 方程(1)：恒定弹性需求模型

$$D(\Delta P_{fare}) = (1 + \Delta P_{fare})^{\varepsilon}$$

**经济含义**：
- 采用经济学标准的**恒定弹性需求函数**（Constant Elasticity Demand Function）
- $\varepsilon < 0$ 表示价格上涨导致需求下降
- 需求因子 $D$ 永远为正，避免了线性模型在大范围涨价时出现负值的失真问题

**与线性模型的对比**：

| 模型 | 公式 | 涨价+100%时的需求因子 |
|------|------|----------------------|
| 线性模型（旧） | $1 + \Delta P_{fare} \times \varepsilon$ | $\max(0.3, 1 + 1 \times (-0.8)) = 0.3$（被截断） |
| 恒定弹性模型（新） | $(1 + \Delta P_{fare})^{\varepsilon}$ | $(1 + 1)^{-0.8} \approx 0.57$（自然衰减） |

**示例**：票价上涨 10%，需求弹性 -0.8
$$D = (1 + 0.1)^{-0.8} = 1.1^{-0.8} \approx 0.927$$
即客流量下降至原来的 92.7%

---

#### 方程(2)：有效燃油成本

$$C_{fuel} = V_{fuel} \times \left[ 1 + \Delta P_{oil} \times (1 - h) \right]$$

**经济含义**：
- 套期保值覆盖比例为 $h$ 的燃油需求，这部分成本被锁定
- 仅 $(1-h)$ 部分暴露于油价波动

**示例**：基础燃油成本 24.70 万亿₫，油价上涨 40%，套保比例 30%
$$C_{fuel} = 24.70 \times [1 + 0.4 \times (1 - 0.3)] = 24.70 \times 1.28 = 31.62 \text{ 万亿₫}$$

---

#### 方程(3)：动态缩放的变动成本

$$V_{other}' = V_{other} \times D(\Delta P_{fare})$$

**经济含义**：
- 变动成本（餐食、地服、航路费等）与客流量成正比
- 当需求下降时，变动成本相应减少，准确反映真实的边际贡献

**关键改进**：修复了旧模型中"变动成本不随客流下降"的Bug

---

#### 方程(4)：调整后收入

$$R_{adj} = R_{base} \times (1 + \Delta P_{fare}) \times D(\Delta P_{fare})$$

代入恒定弹性公式：
$$R_{adj} = R_{base} \times (1 + \Delta P_{fare})^{1 + \varepsilon}$$

**经济含义**：
- 票价调整有直接效应（$(1 + \Delta P_{fare})$）和间接效应（需求因子 $D$）
- 当 $\varepsilon = -0.8$ 时，$(1 + \Delta P_{fare})^{0.2}$ 表示收入对价格的敏感度较低

**示例**：基础收入 64.94 万亿₫，票价上涨 8%
$$R_{adj} = 64.94 \times 1.08^{0.2} \approx 64.94 \times 1.015 = 65.92 \text{ 万亿₫}$$

---

#### 方程(5)：总边际贡献（Contribution Margin）

$$CM = R_{adj} - V_{other}' - C_{fuel}$$

展开形式：
$$CM = R_{base} (1 + \Delta P_{fare})^{1+\varepsilon} - V_{other} (1 + \Delta P_{fare})^{\varepsilon} - V_{fuel} [1 + \Delta P_{oil} (1-h)]$$

**经济含义**：
- $CM > 0$ 是继续运营的必要条件（边际贡献为正）
- $CM \geq FC$ 是实现盈利的充分条件（边际贡献覆盖全部固定成本）

---

#### 方程(6)：预期运营利润

$$\pi = CM \times (1 - \gamma) - FC$$

其中 $\gamma$ 为航班削减比例。

---

### 三阶决策逻辑

模型根据边际贡献（$CM$）与固定成本（$FC$）的关系，将决策状态分为三阶：

```
┌─────────────────────────────────────────────────────────────────────┐
│                        决策状态分类                                 │
├─────────────────────────────────────────────────────────────────────┤
│                                                                     │
│   状态一：运营健康              状态二：现金流承压        状态三：现金流毁灭 │
│   CM ≥ FC                      0 < CM < FC              CM ≤ 0      │
│ ┌─────────────┐            ┌─────────────┐         ┌─────────────┐  │
│ │             │            │             │         │             │  │
│ │    ✅       │            │    ⚠️       │         │    🚨       │  │
│ │             │            │             │         │             │  │
│ │  γ = 0%     │            │   γ = 动态  │         │   γ = 90%   │  │
│ │             │            │             │         │             │  │
│ │  无需削减   │            │  部分削减   │         │  战略性停飞 │  │
│ └─────────────┘            └─────────────┘         └─────────────┘  │
│                                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

#### 状态一：运营健康（$CM \geq FC$）

$$\gamma = 0$$

#### 状态二：现金流承压（$0 < CM < FC$）

保留比例 $= \frac{CM}{FC} \times (1 - \alpha) + \alpha$

$$\gamma = \max(0.05, \min(0.80, 1 - \text{保留比例}))$$

#### 状态三：现金流毁灭（$CM \leq 0$）

$$\gamma = 0.90$$

---

### Phase 1：阈值计算与决策路径

#### 阈值1：票价盈亏平衡点

求解 $CM = 0$ 时的票价调整幅度：

$$R_{base}(1+f)^{1+\varepsilon} - V_{other}(1+f)^{\varepsilon} - V_{fuel}(1+\Delta P_{oil}) = 0$$

使用数值方法（`scipy.optimize.fsolve`）求解。

#### 阈值2：最大可承受油价涨幅

$$\Delta P_{oil}^{max} = \frac{R_{base}(1+f_{max})^{1+\varepsilon} - V_{other}(1+f_{max})^{\varepsilon} - V_{fuel}}{V_{fuel}(1-h_{max})}$$

#### 阈值3：触发削班的临界套保比例

求解 $CM = FC$ 时的套保比例：

$$h^* = 1 - \frac{R_{base}(1+f_{max})^{1+\varepsilon} - V_{other}(1+f_{max})^{\varepsilon} - V_{fuel} - FC}{V_{fuel} \times \Delta P_{oil}}$$

#### 四阶段决策路径

| 阶段 | 油价范围 | 响应动作 | 紧急程度 |
|------|----------|----------|----------|
| 1 | $\le 60\%$ 油价容忍上限 | 监控 | 🟢 低 |
| 2 | 60%~85% 油价容忍上限 | 票价上调 | 🟡 中 |
| 3 | 85%~100% 油价容忍上限 | 套保加仓 + 票价 | 🟠 高 |
| 4 | > 100% 油价容忍上限 | 削班启动 | 🔴 紧急 |

---

### Phase 2：约束型动作序列优化器

#### 优化目标

$$\max_{\Delta P_{fare}, h, \gamma} \quad \pi(\Delta P_{fare}, h, \gamma)$$

**约束条件**：
- $0 \le \Delta P_{fare} \le f_{max}$
- $0 \le h \le h_{max}$
- $0 \le \gamma \le 0.9$

#### 分层优化逻辑

**Step 1：票价寻优**（网格搜索）

$$\max_{\Delta P_{fare}} CM(\Delta P_{fare}, h=0)$$

**Step 2：套保寻优**

若 $CM < FC$，计算覆盖固定成本所需的最小套保比例。

**Step 3：削班判断**

若套保后 $CM < FC$，按三阶逻辑确定削减比例。

### 模块详解

#### `demand_model.py` — 需求模型模块

**职责**：封装恒定弹性需求模型和边际贡献计算逻辑

**核心类/方法**：
| 类/方法 | 功能 |
|---------|------|
| `DemandModel` | 需求模型类 |
| `calculate_demand_factor()` | 计算需求因子 $(1+\Delta P_{fare})^{\varepsilon}$ |
| `calculate_contribution_margin()` | 计算边际贡献（含动态变动成本） |
| `solve_fare_breakeven()` | 求解票价盈亏平衡点 |

**关键输出**：`DemandCalculationResult` 数据类，包含需求因子、调整后收入、变动成本、燃油成本、边际贡献

---

#### `threshold_calculator.py` — 阈值计算模块

**核心类/方法**：
| 类/方法 | 功能 |
|---------|------|
| `ThresholdCalculator` | 阈值计算器 |

## 📊 模型特性

### Phase 1：阈值计算与决策路径

- **🎯 智能决策模式**：输入油价冲击，自动计算最优响应路径
- **📈 阈值看板**：票价盈亏平衡点、油价容忍上限、削班触发套保线
- **🚨 预警系统**：约束违规提示、风险等级标识
- **📍 四阶段路径**：监控 → 票价 → 套保 → 削班

### Phase 2：约束型动作优化器

- **🔢 动作序列优化**：按优先级依次优化票价→套保→削班
- **⚡ 网格搜索**：稳健的全局优化方法
- **💡 推荐理由**：每步动作附带详细经济学解释
- **✅ 可行性检查**：自动检测约束违规情况

---

## ⚠️ 注意事项

1. 本模型为简化分析工具，实际决策需考虑更多因素
2. 削减航班可能扩大账面亏损，但可缓解短期现金流压力
3. 套保具有双刃剑效应，油价下跌时可能产生机会成本
4. 国际航线受双边协定约束，需考虑合规性风险
5. 需求弹性假设基于行业平均值，实际弹性可能因航线、市场而异
